# TodoList-FullStack

## Project Tech Stack
- Frontend
    - iOS / Swift & SwiftUI
    - Website / Next.js / TailwindCSS
- Backend
    - Nest.js
    - MySQL
- Important Tools
    - Vercel (Host Frontend Web & Backend Server)
    - Firebase (Authentication)
